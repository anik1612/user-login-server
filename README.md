# User Login Server

## About this project:

* this is a simple node js application where a user can create an account and login to their account.

## About technology:
* Node JS
* Express JS
* MongoDB 
* Mongoose

### For observe this project 

``` bash
git clone <repo>
npm i
npm run server
```
